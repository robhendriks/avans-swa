# Specification

## Sprite

**Properties**

| Name | Type | Description | Required |
| ---- | ---- | ----------- | -------- |
| id | String | Unique sprite ID | yes |
| image | Object | Image object | yes |
| image.url | String | Image path | yes |

```json
{
  "id": "enemy-a-sprite",
  "image": {
    "url": "/relative/image.jpg"
  }
}
```
